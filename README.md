# Ferramentas

Uma coletânia de ferramentas feitas para ajudar donos de servidores

## Dependências

Lê o plugin.yml dos plugins enviados, para extrair todas as dependencias, e te mostrar quem usa o que.

## FailMark

Ninguem entende direito qual a performance de um Xeon antigo, por isso fiz esse site para comparar um processador com outros parecidos em relação a nota single thread do passmark.

## VirusParcial

Hoje em dia é cada vez mais comum alguem enviando plugin ou mod com virus, então essa parte do website é dedicada a proteger usuários de donos de servidores de virus ja encontrados pela comunidade. Ou seja, ninguem cai na mesma furada duas vezes!
